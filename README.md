# 🎉 La Ruleta de León

Juego de ruleta para el cumpleaños de León (9 años). Gira la ruleta, completa el reto y suma puntos por equipos.

**Jugar en línea:** https://lokimontt.github.io/cumple-leon/

## Segmentos
- ⚽ Fútbol · 🎮 Gamer · 🦖 Animales · 🚀 Ciencia · 🔷 Logos · 🎬 Emojis (trivia)
- 💪 Ejercicio · 🎭 Mímica · 😂 Chistes · 🕺 Baile · 🔍 Ojo de Águila (retos)

## Foto de León
La pantalla de inicio muestra una foto desde el archivo `leon.jpg` (o `.png`).
Si no existe, muestra un emoji 🥳. Para personalizarla, sube tu `leon.jpg` a este repo.
